# 📊 Microsoft AdventureWorks E-Commerce Sales Analysis

## Overview

This project performs an end-to-end Exploratory Data Analysis (EDA) on the Microsoft AdventureWorks E-Commerce dataset. The objective is to uncover sales trends, customer purchasing behavior, product performance, pricing patterns, freight costs, and regional business insights through data-driven analysis.

## Project Objectives

* Clean and preprocess raw sales data
* Handle missing values and data type inconsistencies
* Perform descriptive and statistical analysis
* Analyze sales performance across products and regions
* Evaluate the impact of discounts, freight charges, and promotions
* Generate actionable business insights through visualizations

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Analysis Performed

### Data Preprocessing

* Data cleaning
* Missing value treatment
* Duplicate checking
* Datatype conversion

### Exploratory Data Analysis

* Product Category Analysis
* Regional Sales Analysis
* Order Type Analysis
* Promotion Effectiveness
* Freight Cost Analysis
* Revenue and Net Sales Analysis
* Discount Impact Assessment
* Correlation Analysis

## Key Insights

* Bulk orders generated higher sales than regular orders.
* Significant variations were observed across sales regions.
* Freight charges showed a strong relationship with total customer cost.
* Product categories contributed differently to overall revenue.
* Promotional strategies influenced average net sales performance.
